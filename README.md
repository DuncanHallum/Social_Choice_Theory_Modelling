Modelling part of my 3rd year Individual Maths Project on the Mathematics behind Social choice theory.

Using a GMM to implement a spatial model, which is used to:
- Approximate probabilities of Condorcet cycles occuring.
- Comparing how different voting rules incentivise tactical voting (manipulation).
